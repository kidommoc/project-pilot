# Contract: Documentation Refactor

**Opened**: 2026-04-03T18:00 | **Priority**: high

## Goal
Refactor project-pilot documentation for agent efficiency: SKILL.md becomes a lightweight execution framework; workflow details split by phase; all content unified to English.

## Problem Statement
Current SKILL.md (~600 lines) mixes philosophy, theory, and execution steps, making it hard for agents to find what they need quickly. Chinese content in workflow.md and naming-conventions.md creates inconsistency. Session recovery logic is scattered.

## Proposal
1. Refactor SKILL.md into a compact execution framework (~200-250 lines)
2. Split workflow.md into 4 phase-specific files
3. Create references/contract-guide.md for detailed concepts
4. Translate all Chinese content to English
